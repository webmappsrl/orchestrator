# Tag Group — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Creare un modello `TagGroup` che aggrega dinamicamente Story tramite condizioni booleane sui tag: N gruppi in AND, ciascuno con una lista di tag in OR.

**Architecture:** `TagGroup` è un modello Eloquent separato da `Tag`. Le condizioni sono memorizzate in una tabella pivot `tag_group_conditions` con un campo `group_index` (intero) che identifica il gruppo AND. Ogni gruppo con lo stesso `group_index` è in OR; tutti i gruppi distinti sono in AND tra loro. La relazione `stories()` su `TagGroup` esegue la query dinamica. Nova espone una risorsa con gestione delle condizioni via `HasMany` e una sezione di story matchanti in sola lettura.

**Tech Stack:** Laravel 10, Laravel Nova, PostgreSQL, Eloquent

- [ ] **Step 4: Crea `app/Models/TagGroup.php`**

```php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Builder;
use Illuminate\Database\Eloquent\Collection;
use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;

class TagGroup extends Model
{
    use HasFactory;

    protected $fillable = ['name', 'description'];

    public function conditions()
    {
        return $this->hasMany(TagGroupCondition::class);
    }

    /**
     * Ritorna le Story che matchano tutte le condizioni AND del gruppo.
     * Ogni group_index è un gruppo OR; tutti i gruppi sono in AND.
     */
    public function stories(): Collection
    {
        // Raggruppa i tag_id per group_index
        $groups = $this->conditions()
            ->with('tag')
            ->get()
            ->groupBy('group_index');

        if ($groups->isEmpty()) {
            return new Collection();
        }

        $query = Story::query()->select('stories.*');

        foreach ($groups as $groupIndex => $conditions) {
            $tagIds = $conditions->pluck('tag_id');

            // Ogni AND group: la story deve avere almeno uno dei tag in $tagIds
            $query->whereHas('tags', function (Builder $q) use ($tagIds) {
                $q->whereIn('tags.id', $tagIds);
            });
        }

        return $query->distinct()->get();
    }
}
```

```php
namespace App\Nova;

use Illuminate\Http\Request;
use Laravel\Nova\Fields\HasMany;
use Laravel\Nova\Fields\ID;
use Laravel\Nova\Fields\Text;
use Laravel\Nova\Fields\Textarea;
use Laravel\Nova\Fields\MorphToMany;

class TagGroup extends Resource
{
    public static $model = \App\Models\TagGroup::class;

    public static $title = 'name';

    public static $search = ['id', 'name'];

    public function fields(Request $request): array
    {
        return [
            ID::make()->sortable(),

            Text::make('Nome', 'name')
                ->sortable()
                ->rules('required', 'max:255'),

            Textarea::make('Descrizione', 'description')
                ->nullable()
                ->hideFromIndex(),

            HasMany::make('Condizioni', 'conditions', \App\Nova\TagGroupCondition::class),

            // Story matchanti — sola lettura, solo in dettaglio
            MorphToMany::make('Story', 'storiesViaTagGroup', \App\Nova\Story::class)
                ->onlyOnDetail()
                ->searchable(),
        ];
    }

    public function cards(Request $request): array { return []; }
    public function filters(Request $request): array { return []; }
    public function lenses(Request $request): array { return []; }
    public function actions(Request $request): array { return []; }
}
```

```php
Text::make('Story Matchanti', function () {
    $stories = $this->stories();

    if ($stories->isEmpty()) {
        return '<em>Nessuna story corrisponde alle condizioni.</em>';
    }

    $links = $stories->map(function ($story) {
        $url = url("/nova/resources/stories/{$story->id}");
        $title = e($story->title ?? "Story #{$story->id}");
        return "<a href=\"{$url}\" target=\"_blank\">{$title}</a>";
    })->join('<br>');

    return "<div>{$links}</div>";
})
->asHtml()
->onlyOnDetail()
->readonly(),
```
